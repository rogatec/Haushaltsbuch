# Haushaltsbuch
exercise unit testing and clean code of a simple book of household accounts

[Clean Code Developer School - Application Katas](http://ccd-school.de/coding-dojo/application-katas/haushaltsbuch/)

## How to use

- if running the application in Visual Studio the arguments must be set in the application properties debug settings
- the application needs at least one argument
  - übersicht | einzahlung | auszahlung <-- _as this is a german test it means overview|deposit|payoff_
- other arguments can be seen on the original assignment of the exercise
